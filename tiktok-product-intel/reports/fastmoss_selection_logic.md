
# FastMOSS 爆款产品筛选逻辑

## 一、核心筛选维度

### 1. 基础指标筛选 (商品搜索页面)

| 指标 | 爆款门槛 | 说明 |
|------|----------|------|
| **7天销量** | >5,000 | 近期有热度 |
| **总销量** | >100,000 | 历史验证 |
| **达人出单率** | >50% | 达人愿意带货 |
| **带货达人数** | >100 | 有一定推广规模 |
| **售价区间** | 20-500 THB | 最佳转化区间 |

### 2. 进阶指标

| 指标 | 爆款特征 | 分析价值 |
|------|----------|----------|
| **销量趋势** | 7天/总销量比值 > 5% | 热度上升中 |
| **佣金比例** | 5-20% | 有足够利润空间 |
| **竞争度** | 商品数/店铺数 < 3 | 不是红海 |

### 3. 快速判断方法

**🔥 爆款特征自检表:**
- [ ] 7天销量 > 总销量5% → 热度持续
- [ ] 售价在 50-300 THB → 泰国消费者接受度最高
- [ ] 有3个以上达人带货 → 验证市场
- [ ] 达人出单率 > 60% → 转化效果好

## 二、实操筛选流程

### Step 1: 确定品类
```
商品搜索 → 选择"泰国" → 选择"食品饮料"分类
```

### Step 2: 基础筛选
```
总销量: 100,000以上
近7天销量: 5,000以上
商品状态: 在售
```

### Step 3: 二次验证
```
达人出单率: >50%
带货达人数: >50
```

### Step 4: 利润率计算
```
售价 × 0.3 = 估算出厂价
(售价 - 平台费12% - 运费) / 出厂价 = 利润率
```

| 特征 | 描述 |

| **品类** | 零食/咖啡/干货/调味料 |

## 四、注意事项

1. **防伪冒品**: 泰国对假货处罚严格
2. **食品认证**: 进口食品可能需要FDA认证
3. **季节性**: 榴莲等季节性商品需注意库存
4. **物流**: 食品保质期影响库存周转

## 五、推荐关注品类 (基于已采集数据)

1. **咖啡类** - 佣金高30%, 市场需求稳定
2. **干货类** - 保质期长, 适合海运
3. **饼干零食** - 复购率高
4. **本地特色** - 如榴莲、山竹等热带水果加工品
