# 舆情分析平台重构框架

本目录用于沉淀新产品的需求、信息架构、数据模型和页面线框。这里不沿用旧项目内页逻辑，而是按新的“规则自动打标 + 人工校正 + 规则学习 + 在线报告确认 + 多格式导出”闭环重新设计。

## 当前产物

- `product-blueprint.md`：产品定位、业务流程、模块边界
- `wireframes.md`：六个核心页面低保真线框
- `data-model.md`：核心数据对象、权限、规则学习的数据关系

## 当前版本判断

v1 先解决一个完整项目闭环：

1. 创建项目并选择标签体系、规则版本、报告模板
2. 导入 Excel 并完成字段映射
3. 按规则自动打标签
4. 在在线多维表格中人工校正并自动保存
5. 根据人工修改生成规则学习建议
6. 在线编辑和确认报告
7. 导出 Excel / Word / PDF / PPT

## 视觉方向

- 主色：绿色，用于完成、确认、正常状态
- 辅色：黄色，用于人工确认、待复核、规则建议
- 背景：白色和浅灰白
- 风格：专业、简洁、亲和，偏高级数据报告产品，不做夸张大屏
