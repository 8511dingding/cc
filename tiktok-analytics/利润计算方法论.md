# TikTok 订单利润计算公式

---

## 基础配置

```
THB → CNY:  ÷ 4.7
USD → CNY:  × 7
```

---

## 订单类型判定

```
┌─────────────────────────────────────────────────────────┐
│  "Normal or Pre-order" 字段                              │
├───────────────────────┬─────────────────────────────────┤
│  = "Normal"           │  = 空值 / NaN                   │
├───────────────────────┼─────────────────────────────────┤
│  ▶ Normal 订单        │  ▶ 样品订单 (达人样品)            │
│  ▶ 有收入             │  ▶ 无收入                        │
└───────────────────────┴─────────────────────────────────┘
```

---

## 一、Normal 订单

### 收入
```
         买家实付总额(THB)
收入  =  ─────────────────  ÷  4.7  =  CNY
              1
```

### 成本构成
```
产品成本  =  (出厂价  +  海运物流单件成本)  ×  Quantity
包装成本  =  包装费(单件，不乘数量)
平台佣金  =  买家实付总额  ×  12.29%  +  1.05 THB
包邮服务  =  买家实付总额  ×  7.49%
```

### 平台佣金拆解
```
12.29%  =  3.2% (平台佣金)
              +  8.09% (平台增长费)
              +  1% (汇率损耗)

固定费  =  1.05 THB (每笔订单)
```

### 订单总成本 & 利润
```
订单总成本  =  产品成本  +  包装成本  +  平台佣金  +  包邮服务

利润(CNY)  =  收入  -  订单总成本

利润率(%)  =  利润  ÷  收入  ×  100
```

---

## 二、样品订单 (达人样品)

### 成本构成
```
产品成本  =  (出厂价  +  海运物流单件成本)  ×  Quantity
包装成本  =  包装费(单件，不乘数量)
固定快递费 =  7 CNY (每笔订单，固定)

订单总成本  =  产品成本  +  包装成本  +  固定快递费
```

### 利润
```
收入  =  0

利润  =  0  -  订单总成本  =  -订单总成本

利润率  =  N/A (无收入)
```

## 四、报告输出格式

```
文件: 订单收入大表_YYYYMMDD_timestamp.xlsx

Sheet 1: 总览
  ├─ Normal 订单汇总 (订单数、收入、成本、利润、利润率)
  ├─ 样品订单汇总
  ├─ 全部累加
  ├─ GMV广告 (成本、收入、利润、ROI)
  └─ 项目利润率 (总成本、总利润、利润率)

Sheet 2: 产品统计(Normal)
  └─ 按产品汇总，按订单数降序

Sheet 3: 产品统计(样品)
  └─ 按产品汇总，含快递费

Sheet 4: Normal 订单
  └─ 订单明细

Sheet 5: 样品订单
  └─ 订单明细
```

### 项目利润率计算

```
项目总利润 = 订单利润 + GMV利润
           = 订单利润 + (GMV收入 - GMV成本)

项目总成本 = 订单总成本 + GMV总成本

项目利润率 = 项目总利润 ÷ 项目总成本 × 100%
```

## 五、字段说明

| 字段 | 说明 |
|------|------|
| 出厂价 | 产品成本表中的出厂价 (CNY) |
| 海运物流单件成本 | 从工厂到仓库的物流成本 (CNY) |
| 包装费 | 每个产品的包装材料成本 (CNY) |
| Quantity | 订单中该产品的数量 |
| 买家实付总额 | Order Amount (THB) |
| 12.29% | 平台佣金 3.2% + 增长费 8.09% + 汇率损耗 1% |
| 7.49% | 平台包邮服务费率 |
