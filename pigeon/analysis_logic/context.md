# 客户背景与品牌信息

## 贝亲（Pigeon）品牌背景

### 品牌历史
- 创立于1957年，日本
- 专注于母婴用品研发
- 全球业务覆盖多个国家和地区
- 2012年进入中国市场

### 品牌调性
- 专业、信赖、品质
- 强调"自然实感"核心理念
- 医学级背书

### 市场定位
- 中高端母婴品牌
- 目标人群：注重品质和安全的新手妈妈
- 价格敏感度：中等偏低

## 项目背景

### 分析目的
1. 了解消费者对贝亲产品的真实反馈
2. 识别产品痛点和使用障碍
3. 监控品牌舆情走向
4. 为产品迭代和营销策略提供数据支持

### 分析周期
- 常规报告：每周/每月
- 专项分析：按需

### 数据范围
- 数据来源：抖音平台评论
- 时间范围：按分析周期设定
- 数据量：单次分析不少于500条评论
