### 二、情绪层5分类（取消"负面"小分类）

**旧分类（6类）**：正面、中性、负面、庆幸旁观、恐慌焦虑、愤怒背叛

**新分类（5类）**：
1. **正面**：对a2品牌持信任态度（放心、没问题、官方确认安全）
2. **中性**：多为普通讨论或吃瓜观望
3. **恐慌焦虑**：担忧宝宝健康，害怕中招（如：@豆包 问有没有问题）
4. **庆幸旁观**：庆幸自己没买/已换奶粉；或竞品负面时肯定a2
5. **愤怒背叛**：感觉信任被辜负（如：小作文质疑、代加工质疑）

**合并规则**：
- 原"负面-恐慌焦虑"、"负面-愤怒背叛"、"负面-庆幸旁观"统一归为"负面"大类
- 但在标签呈现时：恐慌焦虑、庆幸旁观、愤怒背叛作为独立子标签
- 实际上5分类的情绪层就是这5个，没有"负面"这个独立分类

### 三、纯@无互动评论的明确定义

**定义**：仅包含@某人，后面无任何有意义内容（去除emoji、空格、标点后为空）

**判断逻辑**：
```
1. 检查是否包含@提及
2. 如果只有一个@，检查@前面的内容是否为空（去除空格）
3. 检查@后面的内容是否为空（去除emoji、空格、标点）
4. 如果@后面只有emoji、空格、标点，则为"纯@无互动"
5. 如果有多个@，则为"纯@无互动"
```

**典型案例**：
- 纯@无互动：`@用户名` 或 `@用户名 [emoji]`
- 非纯@：`@用户名 这下怎么办`（有后续内容）

### 四、认知层4分类（维持不变）

1. **无明确认知**：消费者未表现出明确的认知标签，仍处于观望或被动接收信息状态
2. **信息混淆**：消费者存在信息混淆，分不清国行版、澳洲版、美版的区别
3. **精准认知**：消费者具备精准认知，明白召回范围和特定批次
4. **泛化抵触**：消费者出现泛化抵触或阴谋论倾向

**判断优先级**：
1. 泛化抵触 > 其他（所有奶粉都有问题、企业欺骗等）
2. 精准认知 > 无明确认知（提到FDA、批次、国标等具体信息）
3. 信息混淆 > 无明确认知（分不清版本区别）
4. 默认：无明确认知

### 五、行动层4分类（维持不变）

1. **暂无行动**：无明显行动倾向
2. **寻求帮助**：主动寻求信息或帮助（如：@豆包 求推荐）
3. **转奶流失**：明确表示要换奶粉或已经换奶粉
4. **维权诉求**：提到12315投诉、退款、赔偿等

**判断优先级**：
1. 维权诉求 > 其他（提到12315、赔偿、退款）
2. 转奶流失 > 寻求帮助 > 暂无行动
3. 寻求帮助 > 暂无行动（询问怎么办、哪个安全）

## 认知层分类规则

### 1. 无明确认知
**判断逻辑**：消费者未表现出明确的认知标签

**关键词**：无特定关键词，默认归为无明确认知

### 2. 信息混淆
**判断逻辑**：分不清国行版、澳洲版、美版的区别

**关键词**：
- `国行`、`澳洲版`、`美版`、`版本`
- `分不清`、`搞混`、`区别`
- `到底哪个`、`哪个是正版`

### 3. 精准认知
**判断逻辑**：明白召回范围和特定批次

**关键词**：
- `fda`、`检出`、`仅限美国`
- `特定批次`、`国标`、`配方安全`
- `海关总署`、`检测合格`
- `召回范围`、`官方声明`
- `美版问题`、`区域配方`
- `单点质控`、`美国标准差异`

### 4. 泛化抵触
**判断逻辑**：出现泛化抵触或阴谋论倾向

**关键词**：
- `所有奶粉`、`所有品牌`、`都不行`
- `都有问题`、`都不安全`
- `不如母乳`、`干脆不喝`
- `还是母乳好`、`所有`、`全部`、`一律`
- `阴谋论`、`质疑一切`

**重要区分**：
- **泛化抵触**：提到"所有"、"都"、"所有奶粉/品牌"等扩展性表述
- **负面（plain）**：仅描述自身经历的症状/问题，不扩展到其他品牌
  - 例如"我家喝的a2天天吐"是负面，不是泛化抵触
  - 例如"逮着中国人坑"是对企业的批评指责，属于愤怒背叛

**示例**：
- 泛化抵触：`所有奶粉都有问题，不如喝母乳`
- 负面：`我家喝的a2天天吐，换了[捂脸]`（宝宝症状+担忧 → 恐慌焦虑）
- 愤怒背叛：`资本家太没人性了`（对企业批评 → 愤怒背叛）

## 情绪层判定优先级

1. **愤怒背叛** > 其他：提到企业不负责任、双标、欺骗 → 愤怒背叛
2. **恐慌焦虑** > 中性：担忧宝宝健康、害怕中招 → 恐慌焦虑
3. **正面** > 中性：对a2持信任态度 → 正面
4. **庆幸旁观** > 中性：庆幸自己没买/已换 → 庆幸旁观
5. **中性**：默认分类

**愤怒背叛特殊关键词**：
- 多品牌连续出问题感到崩溃 → 愤怒背叛
- 孩子健康受损责怪企业 → 愤怒背叛
- "气笑了"、"没招了" → 愤怒背叛

## 客户纠正规则应用

根据客户纠正数据（165条 = 97条v8 + 68条0525），以下规则需要特别注意：

### 情绪层
- **宝宝出现症状（如吐奶、拉肚子）+ 担忧情绪 → 恐慌焦虑**（不是负面）
- **庆幸自己没买/已换 → 庆幸旁观**（不是中性）
- **对企业愤怒（如"背刺"、"资本家"）→ 愤怒背叛**（不是负面）
- **多品牌连续中招 → 恐慌焦虑**（不是愤怒背叛，除非明确指责企业）
- **嘲讽进口爱好者、批评崇洋 → 负面**（不是泛化抵触）
- **询问"潜伏期"、"安全吗" → 恐慌焦虑**

### 认知层
- **提到"只喝母乳"、"奶粉都有问题"等绝对化表述 → 泛化抵触**
- **主动研究召回原因、理解美版/国版区别 → 精准认知**
- **混淆a2至初、国版美版等概念 → 信息混淆**
- **只表达情绪但未显示认知程度 → 无明确认知**

### 行动层
- **询问怎么办/怎么处理 → 寻求帮助**（不是暂无行动）
- **表达换奶意向 → 转奶流失**（不是暂无行动）
- **提到12315/赔偿 → 维权诉求**（不是暂无行动）
- **表态"不喝外国奶" → 转奶流失**

### 关键区分
- **泛化抵触 vs 负面**：看是否扩展到全行业/全品牌
- **恐慌焦虑 vs 愤怒背叛**：看是否明确指责企业不负责任
- **信息混淆 vs 精准认知**：看是否理解美版/国版区别

## 计算公式

**占比** = 该分类数量 / 阶段总评论数（第一阶段586，第二阶段4093）

**百分比** = 占比 × 100%，保留1位小数
