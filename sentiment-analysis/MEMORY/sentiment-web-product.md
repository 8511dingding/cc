# 舆情Web产品项目

## 项目概述

新建的Web产品研发项目，用于开发迭代舆情分析在线系统。

## 项目结构

```
sentiment-web-product/
└── web_system/           # Streamlit舆情分析系统
    ├── app.py           # 主应用（v3版本，Google风格UI）
    ├── models.py        # 数据模型
    ├── requirements.txt # 依赖
    └── README.md        # 文档
```

## 技术栈

- **前端**：Streamlit + 自定义CSS（Google风格浅色主题）
- **数据库**：SQLite + SQLAlchemy ORM
- **图表**：Plotly
- **报告生成**：python-docx

## 核心功能

- 多用户认证（admin/user/viewer）
- 数据导入与清洗（6条规则）
- 四层标签体系（认知/情绪/行动/品牌竞品）
- 自动/手动标注
- 报告模板与导出管理

- **官方名**: 飞鹤、a2至初

#### 规则结构

```python
LabelRule:
  layer = "brand"
  stage = "s1" / "s2"
  label = "品牌标准化名称"
  keywords = [官方名, 别名, 俗称, 英文, 拼音, 错拼...]
  priority = 1-40 (市场排名)
  enabled = True
```

#### 当前品牌规则数量

#### 如何编辑品牌规则

1. 登录系统 → 规则管理 → 标签规则
2. 选择层级"品牌竞品"
3. 可查看/编辑每条规则的关键词
4. 支持批量导入JSON格式规则
