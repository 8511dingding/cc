### 四、综合判断逻辑更新

#### 情绪优先级的客户理解
1. **愤怒背叛 > 其他**：提到企业不负责任、双标、欺骗 → 愤怒背叛
2. **恐慌焦虑 > 中性**：提到宝宝症状 + 担忧 → 恐慌焦虑
3. **庆幸旁观 > 中性**：庆幸自己没买/已换 → 庆幸旁观

#### 行动优先级的客户理解
1. **维权诉求 > 转奶流失**：提到投诉、赔偿 → 维权诉求
2. **寻求帮助 > 转奶流失**：询问怎么办 → 寻求帮助
3. **转奶流失 > 暂无行动**：明确表达换奶意向 → 转奶流失

#### @互动类型的特殊处理
- @某人互动 类型中，包含[流泪]等表情 + 担忧内容 → 恐慌焦虑
- @某人互动 类型中，庆幸自己没中招 → 庆幸旁观

#### 4. 重要规则更新

**认知层判定新规则**：
1. 提到"只喝母乳"、"奶粉都有问题"等绝对化表述 → **泛化抵触**
2. 主动研究召回原因、理解美版/国版区别 → **精准认知**
3. 混淆a2至初、国版美版等概念 → **信息混淆**
4. 只表达担忧/愤怒情绪但未显示认知程度 → **无明确认知**

**情绪层判定新规则**：
1. 多品牌连续中招 → **恐慌焦虑**（不是愤怒背叛，除非明确指责企业）
2. "只喝国企/国产" → **庆幸旁观**（不是泛化抵触的情绪部分）
3. 询问"潜伏期"、"安全吗" → **恐慌焦虑**
4. 嘲讽进口爱好者、批评崇洋 → **负面**（不是泛化抵触）

**关键区分**：
- **泛化抵触 vs 负面**：看是否扩展到全行业/全品牌
- **恐慌焦虑 vs 愤怒背叛**：看是否明确指责企业不负责任
- **信息混淆 vs 精准认知**：看是否理解美版/国版区别
