## 标签分类体系

### 认知层（4个分类）
- **无明确认知**：消费者未表现出明确的认知标签，仍处于观望或被动接收信息状态
- **信息混淆**：消费者存在信息混淆，分不清国行版、澳洲版、美版的区别
- **精准认知**：消费者具备精准认知，明白召回范围和特定批次
- **泛化抵触**：消费者出现泛化抵触或阴谋论倾向

### 情绪层（6个分类）
- **正面**：对a2品牌持信任态度（如：放心、没问题、官方确认安全）
- **中性**：多为普通讨论或吃瓜观望
- **负面**：对a2产品产生负面情绪（宝宝喝a2出现问题）
- **庆幸旁观**：庆幸自己没买/已换奶粉；或竞品负面时肯定a2
- **恐慌焦虑**：担忧宝宝健康，害怕中招（如：@豆包 问有没有问题）
- **愤怒背叛**：感觉信任被辜负（如：小作文质疑、代加工质疑）

### 行动层（3个分类）
- **暂无行动**：无明显行动倾向
- **寻求帮助**：主动寻求信息或帮助（如：@豆包 求推荐）
- **转奶流失**：明确表示要换奶粉或已经换奶粉

### 评论内容类型（4个分类）
- **普通内容**：无特殊标记
- **@某人互动**：包含@提及
- **长内容(>100字)**：评论超过100字
- **提及竞品**：提及爱他美、飞鹤等其他品牌

## 报告Word文档结构

### 章节顺序
1. **标题**：a2奶粉召回危机舆情分析报告
2. **一、报告概述**：分析目的、数据概况（时间范围、数据条数）
3. **二、评论内容类型分布**：普通内容/@某人互动/长内容/提及竞品
4. **三、认知分析（Understanding）**：定义 + 两阶段表格 + 解析
5. **四、情绪分析（Emotional）**：定义 + 两阶段表格 + 解析
6. **五、行动层分析（Action）**：两阶段表格
7. **六、结论与建议**

### Word表格格式
每个数据表格包含：
- 表头行：认知类型/情绪类型/行动类型、数量、占比
- 数据行：各分类的具体数值和百分比
- 第二阶段直接在下方继续，不换页

### 数据计算逻辑
1. **阶段数量**：统计评论时间字符串以`2026-04`或`2026-05`开头的记录数
2. **百分比**：`数量 / 阶段总数 * 100`，保留1位小数
3. **品牌提及**：品牌字段以`|`分隔，统计各品牌出现次数

### 解析文本格式
每个分析维度后跟3行解析：
- 第一阶段概况（如：无明确认知为主(67.9%)，精准认知占24.7%）
- 第二阶段概况（如：泛化抵触上升至11.6%）
- 两阶段对比结论（如：第二阶段精准认知比例高于第一阶段）
