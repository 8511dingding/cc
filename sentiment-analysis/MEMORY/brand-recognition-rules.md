# 中国奶粉市场品牌识别规则

## 概述

本规则用于舆情分析系统中自动识别消费者讨论的奶粉品牌。
包含40个品牌，覆盖中国市场销量前30品牌及常见品牌。

## 识别规则结构

```python
{
  "brand": "品牌名",           # 标准化品牌名
  "priority": 1-40,           # 市场销量排名
  "keywords": [...]            # 识别关键词列表
}
```

## 关键词类型说明

| 类型 | 示例 | 说明 |
|------|------|------|
| 官方名 | 飞鹤、a2至初 | 品牌正式名称 |
| 产品系列 | 星飞帆、启赋有机 | 主力产品系列 |
| 俗称 | 蓝臻、德爱、澳爱 | 消费者常用简称 |
| 英文原名 | Aptamil、Enfamil | 英文品牌名 |
| 拼音缩写 | FEIHE、fhh | 拼音首字母 |
| 错别字 | 美赞成 | 常见输入错误 |
| 谐音/变体 | - | 消费者习惯性称呼 |

## 品牌识别优先级说明

| 优先级范围 | 梯队 | 说明 |
|------------|------|------|
| 1-10 | 第一梯队 | 高端/进口品牌，市场销量最大 |

## 使用场景

1. **舆情监测**: 识别消费者讨论的品牌

3. **市场份额**: 估算品牌提及率
4. **口碑分析**: 分析特定品牌的正负面评价
