## 数据清洗规则

1. **完全无效内容需过滤**：
   - 纯乱码（无法辨认的文字）
   - 纯表情符号堆砌
   - 空内容或仅空白字符

2. **品牌/产品识别优先级**：
   - 优先识别全称，其次识别简称
   - 注意上下文区分同品牌不同产品
   - 竞品提及需准确归类到对应品牌

3. **情感分析注意事项**：
   - 注意反讽表达（如"挺好的，买了后悔"）
   - 注意双重否定（如"不便宜"）
   - 专业术语需结合上下文理解
