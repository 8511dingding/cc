# a2舆情分析 - 分类标准手册

> 本文档记录a2奶粉舆情分析的分类标准，基于客户确认的规则和标注样本
> 最后更新：2026-05-24

---

## 核心原则：负面必须针对a2品牌

**判断负面言论的标准**：只有对a2品牌/产品本身产生负面影响的评论才算负面。

- 评论中提及竞品（如爱他美、贝巴/雀巢）负面，不算a2负面，反而说明用户对a2持正面态度
- 用户为a2辩护、表示自己孩子没事、庆幸换了a2等内容，属于庆幸旁观/中性

## 一、认知层分类标准（4种）

| 认知类型 | 判断规则 | 示例 |
|---------|---------|------|
| **泛化抵触** | 对a2品牌全盘否定、阴谋论、攻击品牌 | "a2破产后被华人收购各种造谣"、"所有奶粉都有问题不如喝母乳" |
| **信息混淆** | 分不清a2不同产品/版本差异 | "a2至初和紫白金有什么区别" |
| **精准认知** | 了解召回范围仅限美国特定批次 | "召回仅限美国市场3个批次"、"官方确认安全" |
| **无明确认知** | 中性询问、观望态度 | "a2还能继续喝吗" |

## 二、情绪层分类标准（5种）

| 情绪类型 | 判断规则 | 示例 |
|---------|---------|------|
| **正面** | 对a2品牌/产品持信任和正面态度（包括之前的"认可"） | "娃从出生就喝这款奶粉"、"官方确认安全"、"可以继续喝"、"放心"、"没问题" |
| **中性** | 普通讨论、吃瓜观望 | "a2和a2至初有区别吗"、"求助问问大家" |
| **负面** | 对a2产品产生负面情绪（宝宝喝a2出现问题） | "拉肚子"、"呕吐"、"吐奶"、"腹泻"、"崩溃" |
| **庆幸旁观** | 庆幸自己没买/已换奶粉；或为a2辩护；竞品负面时肯定a2；表达"国产更好" | "还好没买"、"奶粉还得国产"、"不喝外国奶"、"两个娃都喝母乳" |
| **恐慌焦虑** | 担忧宝宝健康，害怕中招 | "a2至初没事吧 瑟瑟发抖"、"会不会有事"、"好担心"、"还能喝不" |
| **愤怒背叛** | 感觉信任被辜负、品牌失望 | "a2该死"、"气死了被骗了"、"失望透顶"、"骗人是狗"、"谁还敢喝" |

**说明**：原"认可"类别已合并到"正面"中。"认可"的判断关键词如"放心"、"没有问题"、"官方确认安全"等现在都归类为"正面"。

**关键区分**：
- 竞品负面（爱他美拉肚子、贝巴不长肉）= a2正面信号 → 庆幸旁观
- a2产品问题（宝宝喝a2拉肚子、绿便）= a2负面 → 负面
- **小作文类评论**（长篇分析论证）= 绝对负面，情绪愤怒，品牌失望

## 六、分类判断优先级

当一条评论同时满足多个条件时，按以下优先级判断：

1. **愤怒背叛 > 其他情绪**
   - 出现"该死"、"欺骗"、"造假"、"品牌失望"等词 → 泛化抵触 + 愤怒背叛
   - 小作文类评论（长篇质疑分析）→ 绝对负面，情绪愤怒，品牌失望

2. **恐慌焦虑 > 中性**
   - 出现"会不会有事"、"好担心"、"还能喝不"等 → 恐慌焦虑

3. **正面 > 中性**
   - 出现"放心"、"没问题"、"官方确认安全"、"可以继续喝" → 正面
   - 在a2内容下对产品表达认可 → 正面

4. **庆幸旁观判定**
   - 竞品负面时 → 庆幸旁观
   - 庆幸没买时 → 庆幸旁观
   - 表达"国产更好"、"喝母乳" → 庆幸旁观

5. **负面判定**
   - a2产品问题（拉肚子/呕吐/吐奶/绿便）→ 负面
   - 明确说"换了"、"转奶" → 转奶流失
   - 长内容表达紧张/吐槽 → 负面

## 八、特殊规则

### 8.1 竞品负面 = a2正面
当评论提到竞品出现问题时（如：爱他美拉肚子、飞鹤又吐又拉），说明用户对a2持正面态度。

### 8.2 "放心""没问题" → 正面
评论明确表示"放心"、"没有问题"、"可以继续喝"时，情绪归类为正面。

### 8.3 长内容大多负面
字符数超过100的长内容评论，大多是表达紧张、吐槽或负面情绪，需重点关注。
**小作文类评论**（长篇质疑分析）= 绝对负面，情绪愤怒，品牌失望

### 8.4 问有没有问题 → 恐慌焦虑
提到a2然后问"有没有问题"、"会不会有事"、"还能喝不"，属于恐慌焦虑情绪。

### 8.5 A2蛋白质 ≠ a2品牌
"A2"单独出现时可能是指"A2蛋白质"成分，需要结合语境判断。
如"君乐宝A2奶源没事吧"中的"A2"是指蛋白质，不是a2品牌。

### 8.6 "国产更好""喝母乳" → 庆幸旁观
表达庆幸选择国产奶粉或母乳时，属于庆幸旁观。

### 8.7 在a2内容下表达认可 → 正面
在a2视频/内容下对产品表达认可（如：我娃从出生就喝这款奶粉），属于正面情绪。

### 8.8 呕吐/拉肚子/吐奶 → 负面
明确提到宝宝喝a2后出现呕吐、拉肚子、吐奶等症状，属于负面情绪。

### 8.9 客服不搭理/敷衍 → 愤怒背叛
提到反馈客服但被敷衍的情况，属于愤怒背叛情绪。

### 8.10 具体负面 ≠ 泛化抵触
对a2的具体批评（如"性价比不高"、"美国市场占有率低"、"化不开"）是负面但不是泛化抵触。
泛化抵触是"所有奶粉都有问题"、"不如喝母乳"这类全盘否定。

### 8.11 "吐"、"干呕"、"爱吐" → 负面
提到宝宝喝a2后出现呕吐相关症状，属于负面情绪。
